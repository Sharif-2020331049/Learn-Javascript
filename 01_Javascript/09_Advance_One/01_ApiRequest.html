readyState Values

0 (UNSENT): The XMLHttpRequest object has been created, but the open() method hasn't been called yet.

1 (OPENED): The open() method has been called, but the request hasn't been sent yet with send().

2 (HEADERS_RECEIVED): The send() method has been called, and the headers of the response and the status are available.

3 (LOADING): The response body is being received (some data has been received but the response is not yet complete).

4 (DONE): The request is complete, and the response is fully received.
